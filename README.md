# AI Tools Directory

정적 사이트(`index.html`)와 Vite+React 앱(`react-ai-tools`)이 함께 있는 프로젝트입니다.

## React 개발 서버

```bash
cd react-ai-tools
npm install
npm run dev
```

## GitHub에 올리기

1. GitHub에서 빈 저장소를 만든 뒤 URL을 복사합니다.
2. 아래에서 `YOUR_USER` / `YOUR_REPO` 를 바꿔 실행합니다.

```bash
git remote add origin https://github.com/YOUR_USER/YOUR_REPO.git
git branch -M main
git push -u origin main
```

처음 푸시 시 로그인 창(PAT 권장)이 뜰 수 있습니다.
